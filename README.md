## FinBot Desktop – AI‑ассистент специализирующийся на анализе актуальных финансовых данных

Понимает запросы на естественном русском языке, сам определяет актив (валюты, акции РФ/США, криптовалюты) и собирает нужные данные: курсы из ЦБ РФ, MOEX, Yahoo Finance и CoinGecko, а также статистику (ставка ФРС, ключевая ставка ЦБ, инфляция).

Параллельно с ценами бот ищет глобальные и свежие новости через DuckDuckGo и Google News RSS, кэширует все ответы для мгновенной работы при повторных вопросах.

Анализ генерируется бесплатной LLM‑моделью OpenRouter (openai/gpt‑oss‑120b), которая учитывает все найденные данные, новости и тренды.

Приложение запускается как обычная программа (можно закрепить на панели задач) и не требует браузера – достаточно Python и ключа OpenRouter.
