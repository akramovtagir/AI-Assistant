# AI-Assistant
Финансовый ассистент специализированный на анализе актуальных финансовых данных
# Текущие возможности:
Бот самостоятельно определяет актив (валюты, криптовалюты, акции), получает актуальные цены через API, ищет свежие новости и глобальный геополитический контекст,
анализирует исторический тренд за 30 дней и выдаёт структурированный финансовый анализ с учётом текущей даты и географии событий.
